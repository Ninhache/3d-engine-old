void UserParameters::render ( Scene *  scene)
virtual

Draws the GUI.

the whole structure of a GUI is defined in the render function and is always similar :

A Begin method : (takes the windows name as argument and a boolean to handle close event)
ImGui::Begin("Window's name", &bool);

A Body (optional): Can be many things (see the Dear ImGui docs for more "things")
ImGui::Text("Hello world!");

An end method :
ImGui::End();

Note
you need as many Begin methods as end methods.
